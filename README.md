# Интернет магазин

## Описание
Проект интернет магазин - Django-проект интернет-магазина
В проекте осуществлены приложения Catalog, Blog
Catalog - приложение Catalog отвечает за каталог товаров разбитых по категориям.
Blog - приложение Blog отвечает за блог магазина.
Созданы функции для управления пользователем.
Созданы функции аутентификации и авторизации а так же обеспечен контроль доступа к определенным частям приложения 
(создание, удаление, редактирование Продукта) в приложении Catalog.

## Установка
- [клонируйте репозиторий](https://github.com/AlexJharinov/market_prj)

## Запуск программы
Запустить проект python manage.py runserver 
Открыть браузер и перейти на http://127.0.0.1:8000/catalog/
